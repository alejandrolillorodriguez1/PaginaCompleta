# Página Completa

Aplicación web completa desarrollada como práctica de desarrollo full stack, utilizando JavaScript, EJS, CSS y Node.js.

## Descripción

Este proyecto consiste en una página web dinámica con estructura de servidor, vistas renderizadas y estilos personalizados. El objetivo principal fue practicar la creación de una aplicación web completa, separando la lógica del servidor, las vistas y los recursos estáticos.

## Tecnologías utilizadas

- JavaScript
- Node.js
- Express
- EJS
- CSS
- HTML

## Funcionalidades

- Estructura de aplicación web completa
- Renderizado de vistas con EJS
- Gestión de rutas desde el servidor
- Uso de archivos estáticos
- Diseño personalizado con CSS

## Objetivo del proyecto

El objetivo de este repositorio es reforzar conocimientos de desarrollo web, especialmente la parte de backend básico con Node.js y la generación de páginas dinámicas.
